## 🧑🏻‍💻 Shimmy
---

블로그 유목민 시미의 개발 블로그입니다.<br>
React, Node.js와 네트워크 등 javascript와 웹에 관한 지식을 기록하고, 나누고 싶습니다.<br>
깃허브 블로그는 또 어떤 맛이 날까요. 열심히 기록하도록 하겠습니다 :)

## 💪 Stacks
---

### ⚙️ 기술 스택

**Front-end**

- Javascript, Typescript
- React.js, Next.js, React-native
- Tanstack query, Storybook

**Back-end**

- Nest.js, Express.js, redis, ElastiCache
- Java, Spring, Spring Boot, MyBatis

### 🛠️ 개발 역량

**Front-end**

- **React**, **Next.js**를 활용한 프로젝트 경험
- 협업에 용이한 **TypeScript** 활용 능력
- **React-native**로 iOS 정책을 준수하는 앱 개발
- PG결제 기능 구현 경험

**Back-end**

- **Express.js**, **Nest.js**, **Spring Boot** 환경에서의 서버 구현
- Node.js 기반 서버 **OAuth, 인증 미들웨어 구현**
- **ElastiCache** 세션 구성 및 **redis** 활용 경험

### 👥 협업 역량

- 리드 개발자 역임: **Github-flow** 전략 채택, 커밋 컨벤션 정립
- **Prettier**, **Lint**, **Husky** 설정으로 코드 컨벤션 준수
- **Jira**를 이용한 애자일 방식 개발

### 🚀 배포 및 운영

- React-native 앱의 Testflight 빌드/배포 및 **실제 앱스토어 런칭**
- **GitHub Action**으로 레포지토리 간 push 연동
- **Vercel**을 이용한 웹 배포
- **AWS EC2**와 RDS를 활용한 서버 구축 및 배포
- SSL 인증과 Nginx 리버스 프록시 설정으로 **HTTPS** 환경 구현
